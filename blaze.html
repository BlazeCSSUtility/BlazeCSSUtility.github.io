What is Blaze CSS?

Blaze CSS is a CSS utility class library that shortens the process of styling elements directly through HTML.

How can I get Blaze CSS?

1. Go to my and download all files.
2. Make sure to read the Read Me (in files and bottom of page).
3. Open the files for your project and the downloaded files side-by-side.
